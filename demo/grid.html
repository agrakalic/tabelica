Simple Grid System

To change the gutter width, simply go to src/scss/layout/grid.scss and edit the variable named $gutter. Please use percentage values.
Unlike many other similar projects, grid system in Tabelica uses margins instead of paddings for columns, reducing the need of an extra child element.
